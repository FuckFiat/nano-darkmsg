# 🌑 NANO Dark Messenger

**Приватность — не роскошь, а базовое право.**

Кроссплатформенный мессенджер с end-to-end шифрованием, работающий через любые каналы связи (SMTP, облака, P2P) даже в условиях "белых списков".

---

## 🔥 Особенности

- **AES-256-GCM** + **EdDSA (X25519)** — современная криптография
- **Работает офлайн** — Bluetooth, Wi-Fi Direct, mesh-сети
- **Стеганография** — шифр в картинках с котиками 🐱
- **Автообмен ключами** — QR-коды, без сложных настроек
- **Self-destruct** — сообщения уничтожаются после прочтения
- **Plausible deniability** — второй пароль для "фейкового" контента

## 🚀 Быстрый старт

### Прототип (Python)
```bash
cd scripts
python3 proto_encrypt.py --help
```

### Сборка ядра (Rust)
```bash
cd core
cargo build --release
```

## 🛡️ Безопасность

- Все ключи хранятся **локально** (Keychain/Keystore)
- Никаких серверов, никаких метаданных
- Open Source — любой может аудить код
- **Не требует лицензию ФСБ** (физлица, личное использование)
